# 100 Days of Code — Data Edition 📊

Welcome to my **#100DaysOfCode** journey! This repo documents 100 days of consistent learning and practice, with a focus on **data analysis, Excel, and data visualization**.

## 🎯 Goal

To build strong, practical data skills by:
- Mastering the **fundamentals of programming** (logic, syntax, problem-solving)
- Learning **Excel** for data cleaning, exploration, and reporting
- Progressing into **Python for data analysis** (pandas, numpy)
- Building **data visualizations** (matplotlib, seaborn, Excel charts, etc.)
- Completing small projects along the way to apply what I learn

| Phase | Folder | Focus | Approx. Days |

| 1 | `python_fundamentals/` | Variables, input/output, loops, functions, data structures | Day 1 – 15 |
